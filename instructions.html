File Upload Requirements ✨

Please upload an Excel (.xlsx, .xls) or CSV file containing sales data.

Required Columns

  • Date
  • Product
  • Quantity
  • Price
  • Cost

Accepted Alternative Column Names

  • Product: Product Name, Item, Item Name
  • Quantity: Qty, Qnty, Units Sold
  • Price: Unit Price, Amount, Rate
  • Date: Transaction Date, Sale Date, Order Date
  • Cost: Unit Cost

Example Format

Date Product Quantity Price Cost

2026-12-01 HP Pavilion Gaming Desktop PC 1 300000 100000
